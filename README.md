# Vue Twitter Training App

# Quickstart

### Install

Clone Vue-Twitter-App project:
```
https://github.com/CuneytYilmaz/vue-twitter-app
```

Install project dependencies via npm:
```
npm install
```

Start the development server:
```
npm start
```
